## Component Specifications

### Header Bar (Both Options)
- Height: 56px
- Elements: Logo | Back arrow | Page title | Client dropdown | Type badge | Action buttons
- Client dropdown shows: "Name — Family (Type)"
- Badge: Green "Impact Client" or Blue "Traditional Client"

### Left Rail (Option A Only)
- Width: 200px fixed
- Sticky on scroll
- Contains: Client name/family, AUM, location, entity type, tax status, advisor, survey date
- Bottom: Quick action buttons

### Card Grid (Option B Only)
- 3 equal-width cards in top row
- Card height: ~180px
- Responsive: stack on mobile

### Hero Section
- Full width
- Impact: Archetype name + match % bar + description
- Non-Impact: Risk profile name + growth/stability/tail metrics

### Content Sections
- Max-width: 1200px centered
- Section headers: 18px Georgia
- Consistent padding: 24px

### Survey Responses
- Collapsible by default
- Category filter pills
- Scrollable container (max-height: 500px)

- Stability: #0B6D7B (teal)

## Interaction Notes

1. **Client Switching**: Instant re-render, no page reload
2. **Survey Expand**: Smooth height animation (300ms)
3. **Category Filter**: Instant filter, scroll to top
4. **AI Launch**: Opens in new tab with context params
5. **Export IPS**: Loading state on button, downloads DOCX

---

## Responsive Breakpoints

- Desktop: 1200px+ (full layout)
- Tablet: 768-1199px (stack cards vertically, hide left rail)
- Mobile: <768px (single column, collapsible sections)
